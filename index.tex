\chapter{Setup}\label{setup}

Voor we aan de slag kunnen met kwantitatief onderzoek in R, moeten we
alle benodigde software en libraries installeren. Rstudio is het
programma waarin we de analyses draaien en resultaten opvragne. Github
is zogenaamde versiebeheer software waar dit handboek open-source gehost
is. Het voordeel daaarvan is dat het boek open toegangelijk is,
makkelijk te updaten en voor jullie makkelijk om mee te werken tijdens
de lessen.

Om het werken met dit boek te vergemakkelijken, kun je het boek in jouw
R inladen zodat je de code en bijbehorende data makkelijk kunt gebruiken
voor de opdrachten tijdens de werkgroepen.

\textbf{Software:}

\begin{itemize}
\item
  \href{https://posit.co/download/rstudio-desktop/}{Rstudio en R}
\item
  \href{https://desktop.github.com/download/}{Github Desktop}
\end{itemize}
